## ¿Qué aprenderé?

Con este curso crearás, optimizarás, automatizarás o mejorarás con los siguientes proyectos:

- 🚀  Implementarás pipelines CI/CD con análisis de vulnerabilidades automatizado (SAST/DAST).
- 🚀  Desarrollarás aplicaciones con prácticas de secure coding y threat modeling.
- 🚀  Configurarás sistemas de protección de aplicaciones web (WAF, DDoS, rate limiting).
