## ¿Qué aprenderé?

Con este curso integrarás prácticas de Ciberseguridad en tu flujo de desarrollo de software:

- 🚀 Implementarás pre-hooks para asegurar tus desarrollos.
- 🚀 Generar Software Bill of Materials (SBOM) e identificar librería principales y transitivas.
- 🚀 Identificación de vulnerabilidad Static Application Security Testing SAST y Secret Scanning localmente.
- 🚀 Configurarás escaneo de dependencias y detección de vulnerabilidades.
- 🚀 Aplicarás principios de codificación segura (OWASP Top 10).
- 🚀 Gestionarás secretos y credenciales de forma segura en tu código.
- 🚀 Implementarás validación y sanitización de entradas de usuario.
- 🚀 Configurarás autenticación y autorización seguras en tus aplicaciones.
- 🚀 Realizarás pruebas de seguridad automatizadas en tu código.
- 🚀 Aprenderás a manejar y reportar vulnerabilidades responsablemente.
